Numbers in JavaScript

Unlike many programming languages, JavaScript has only one type of number. It can represent integers, decimals (floats), and even special values.

Integers

42 -17 0

Decimals

3.14 -0.5 99.99

Scientific

1.5e6 2e-4 5.67e10

Special

Infinity -Infinity NaN
