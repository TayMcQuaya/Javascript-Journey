Module 2: Variables & Data Types

Variables are the foundation of programming - they let you store and manipulate data. In this module, you'll learn how to create variables, understand different data types, and master the art of working with data in JavaScript.

Learning Objectives

  • Understand what variables are and why we need them
  • Learn the differences between let, const, and var
  • Master JavaScript's primitive data types
  • Work with numbers, strings, booleans, null, and undefined
  • Convert between different data types
  • Follow best practices for naming variables
