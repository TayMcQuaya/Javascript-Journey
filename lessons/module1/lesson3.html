Let's Start Coding!

It's time to write your first JavaScript code! We'll start with the traditional "Hello, World!" program and then explore different ways to run JavaScript code.

Method 1: JavaScript in HTML

The easiest way to start is by adding JavaScript directly to an HTML file:

index.html 
<!DOCTYPE html>
<html>
<head>
    <title>My First JavaScript</title>
</head>
<body>
    <h1>Welcome to JavaScript!</h1>
    
    <script>
        // This is your first JavaScript code!
        console.log("Hello, World!");
        
        // Let's show an alert
        alert("Welcome to JavaScript!");
    </script>
</body>
</html>
Note: JavaScript goes inside <script> tags. You can place these tags in the <head> or <body> (usually at the end of body is best).

Method 2: External JavaScript File

For larger projects, it's better to keep JavaScript in separate files:

index.html 
<!DOCTYPE html>
<html>
<head>
    <title>My JavaScript App</title>
</head>
<body>
    <h1>My JavaScript App</h1>
    
    <!-- Link to external JavaScript file -->
    <script src="script.js"></script>
</body>
</html>
script.js 
// This is script.js
console.log("JavaScript loaded from external file!");

// You can write all your JavaScript here
let message = "Hello from script.js!";
console.log(message);
