Lesson 1: What is JavaScript?

Understanding the world's most popular programming language

Introduction

JavaScript is a high-level, interpreted programming language that makes websites interactive. Created in 1995 by Brendan Eich in just 10 days, JavaScript has evolved from a simple scripting language to one of the most powerful and versatile programming languages in the world.

Fun Fact: Despite its name, JavaScript has no relation to Java. It was originally called "LiveScript" but was renamed for marketing reasons.

Why Learn JavaScript?

Runs Everywhere

JavaScript runs in browsers, servers (Node.js), mobile apps, and even IoT devices.

Huge Community

Millions of developers use JavaScript, creating a vast ecosystem of libraries and tools.

Career Opportunities

JavaScript developers are in high demand with excellent job prospects.

Easy to Start

You only need a web browser to start coding in JavaScript!

What Can You Build with JavaScript?

  • Interactive Websites: Add dynamic content, animations, and user interactions
  • Web Applications: Build complex apps like Gmail, Google Maps, or Facebook
  • Mobile Apps: Create native mobile apps using React Native or Ionic
  • Server Applications: Build backend services with Node.js
  • Desktop Applications: Create desktop apps with Electron (VS Code is built with it!)
  • Games: Develop 2D and 3D games that run in the browser
  • Machine Learning: Use TensorFlow.js for AI in the browser

Your First Look at JavaScript

Here's what JavaScript code looks like:

example.js 
// This is a comment - it doesn't run
console.log("Hello, World!");

// Variables store data
let message = "Welcome to JavaScript!";
let year = 2024;

// Functions perform actions
function greet(name) {
    return "Hello, " + name + "!";
}

// Using the function
console.log(greet("Student"));
