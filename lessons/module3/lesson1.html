Learn to perform mathematical operations in JavaScript

Introduction

Arithmetic operators are like the basic math operations you learned in school, but now you're using them to tell a computer how to calculate things! Just like a calculator, JavaScript can add, subtract, multiply, divide, and more.

Think of arithmetic operators as the mathematical tools in your programming toolbox. Whether you're calculating a shopping cart total, determining game scores, or analyzing data, you'll use these operators constantly.

Basic Arithmetic Operators

JavaScript has five basic arithmetic operators that work just like regular math:

The Basic Five:
  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)

The Modulus Operator (%)

                
The modulus operator might be new to you. It gives you the remainder after division. This is incredibly useful for many programming tasks!


                

                    
 Real-World Example: Egg Cartons

                    
If you have 17 eggs and egg cartons hold 6 eggs each, how many eggs are left over?

                    
17 % 6 = 5 (You'll have 5 eggs that don't fit in a complete carton)

Common Mistakes to Avoid

                

                    

                        
                        

                              Forgetting Order of Operations
                            
Always remember PEMDAS! Use parentheses when in doubt.

                        

                    

                    

                        
                        

                              Division by Zero
                            
Dividing by zero gives Infinity in JavaScript, not an error!

                        

                    

                    

                        

                              String Concatenation vs Addition
                            
Remember: '5' + 3 = '53' not 8!

                        

                    

                

            


            

                
Summary

                

                    
Key Takeaways:

                    
    
                        
  • JavaScript has 5 basic arithmetic operators: +, -, *, /, %
    • 
                        
  • The modulus operator (%) gives you the remainder after division
    • 
                        
  • JavaScript follows PEMDAS order of operations
    • 
                        
  • The ** operator is used for exponentiation
    • 
                        
  • Be careful with the + operator and strings - it concatenates!
